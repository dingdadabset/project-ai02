# 割草游戏（草原风暴）项目开发台账 - 总览

## 一、项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 草原风暴 (Grass Storm) |
| 项目类型 | 割草类动作游戏 (Horde Survival) |
| 目标平台 | iOS / Android 移动端 |
| 技术栈 | Cocos Creator 3.8+ / TypeScript |
| 仓库地址 | dingdadabset/project-ai02 |
| 预计周期 | 8 周（MVP 2周 → Alpha 4周 → Beta 6周 → Release 8周） |

---

## 二、模块总览

本项目共分为 **8 大模块**，**46 个功能任务**：

| 编号 | 模块名称 | 子任务数 | 阶段 | 优先级 |
|------|----------|----------|------|--------|
| M01 | 核心框架 (Core) | 6 | MVP | P0 |
| M02 | 玩家系统 (Player) | 5 | MVP | P0 |
| M03 | 敌人系统 (Enemy) | 5 | MVP | P0 |
| M04 | 武器系统 (Weapon) | 6 | MVP/Alpha | P0 |
| M05 | 战斗系统 (Combat) | 5 | MVP | P0 |
| M06 | 关卡系统 (Level) | 5 | Alpha | P1 |
| M07 | UI 系统 (UI) | 7 | MVP/Alpha | P1 |
| M08 | 养成与商业化 (Meta) | 7 | Beta/Release | P2 |

---

## 三、开发阶段划分

### Phase 1: MVP（第 1-2 周）
**目标**：可玩原型——角色能移动、能攻击、能升级

涉及模块：M01 核心框架 + M02 玩家 + M03 敌人（基础）+ M04 武器（1个）+ M05 战斗 + M07 UI（基础）

### Phase 2: Alpha（第 3-4 周）
**目标**：核心循环完整——4 种武器 + 技能系统 + 关卡 + Boss

涉及模块：M04 武器（扩展）+ M06 关卡 + M03 敌人（扩展）+ M07 UI（完善）

### Phase 3: Beta（第 5-6 周）
**目标**：完整体验——养成系统 + 装备 + 多关卡 + 音效美术

涉及模块：M08 养成 + M07 UI（全部）+ 美术/音效接入

### Phase 4: Release（第 7-8 周）
**目标**：上线版本——性能优化 + SDK 接入 + 真机测试

涉及模块：性能优化 + 商业化 + 测试

## 四、模块依赖关系图

```
M01 核心框架
 ├── M02 玩家系统 ──────┐
 ├── M03 敌人系统 ──────┤
 │                      ▼
 ├── M04 武器系统 ──→ M05 战斗系统
 │                      │
 ├── M06 关卡系统 ◄─────┘
 │      │
 │      ▼
 ├── M07 UI 系统
 │      │
 │      ▼
 └── M08 养成与商业化
```

**关键依赖说明**：
- M01 是所有模块的基础，必须最先完成
- M02/M03 依赖 M01，且互相独立可并行
- M04 依赖 M02（武器挂载在玩家上）
- M05 依赖 M02 + M03 + M04（战斗涉及三者交互）
- M06 依赖 M03 + M05（关卡调度刷怪和战斗）
- M07 依赖 M02 + M05 + M06（UI 显示游戏数据）
- M08 依赖 M06 + M07（养成建立在关卡通关之上）

## 六、接口（事件通信）一览

本项目采用事件总线解耦，核心事件接口如下：

| 事件名 | 触发方 | 消费方 | 数据载荷 |
|--------|--------|--------|----------|
| ENEMY_KILLED | Enemy | CombatSystem, DropSystem, HUD, GameManager | {enemy, position, exp, dropTable} |
| ENEMY_DAMAGED | Enemy | DamageNumber | {enemy, damage, isCrit, position} |
| PLAYER_DAMAGED | Player | HUD, GameManager | (damage, currentHp) |
| PLAYER_DEAD | Player | GameManager, LevelSystem | - |
| PLAYER_HEAL | Player | HUD, DamageNumber | (amount) |
| EXP_GAINED | Player | HUD | (currentExp, expToNext) |
| PLAYER_LEVEL_UP | Player | GameManager, SkillSystem, HUD, SkillSelect | (level) |
| SKILL_SELECTED | SkillSystem | SkillSelect, GameManager | (SkillOption) |
| WAVE_START | SpawnSystem | HUD | (waveIndex) |
| WAVE_COMPLETE | SpawnSystem | LevelSystem | - |
| BOSS_SPAWN | SpawnSystem | GameManager, HUD | (enemyId) |
| LEVEL_COMPLETE | LevelSystem/SpawnSystem | GameManager | - |
| GAME_OVER | GameManager | LevelSystem, UI | - |
| GAME_PAUSE | GameManager | All Systems | - |
| GAME_RESUME | GameManager | All Systems | - |
| GAME_START | GameManager | All Systems | - |
| ITEM_PICKED | DropSystem | HUD | (type, value) |
| ITEM_DROPPED | DropSystem | - | (type, position) |
| GOLD_GAINED | DropSystem | HUD, SaveManager | (amount) |
| WEAPON_ACQUIRED | Player/SkillSystem | HUD | (weaponId) |
| WEAPON_UPGRADE | SkillSystem | WeaponBase | (weaponId, level) |
