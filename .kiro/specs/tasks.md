## 任务优先级说明

| 优先级 | 标识 | 说明 |
|--------|------|------|
| P0 | 🔴 | 阻塞性任务，必须立即完成 |
| P1 | 🟡 | 核心功能，当前 Sprint 完成 |
| P2 | 🟢 | 重要但不紧急，可延后 |
| P3 | ⚪ | 锦上添花，资源充裕时做 |
